->Para orgazinar, crie uma pasta

mkdir teste1
cd teste1

-> Para rodar algo no DaVinci, foi preciso de 2 arquivos diferentes:
   -> Um arquivo (.yaml) onde irá conter informações dos seus dados, se é MC, qual tipo de dados, quantidade máxima de eventos...
   -> Um arquivo (.py) onde irá conter uma função que fará deste a seleção das variáveis até a resconstrução dos seus objetos

->Feito esses arquivos, para rodar utilizei a seguinte linha de comando:

lb-run -c x86_64_v2-el9-gcc13-opt DaVinci/v66r3 lbexec (NOME DO MEU ARQUIVO .py) :(NOME DA FUNÇÃO PRESENTE NO .py) (MEU ARQUIVO .yaml).yaml
